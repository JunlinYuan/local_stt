---
description: Generate and open the iOS Xcode project for installing on iPhone
---

Run `xcodegen generate` in the `ios/` directory to generate the Xcode project, then open `ios/LocalSTT.xcodeproj` in Xcode.

After opening, remind the user to:
1. Select their iPhone as the run destination
2. Verify signing team under Signing & Capabilities
3. Press Cmd+R to build and install
